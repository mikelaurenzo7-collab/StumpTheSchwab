# StumpTheSchwab

A web-based music production platform built to make it easier to produce higher quality music.

## Features

- **Step Sequencer** — 8-track grid with 8/16/32/64 step patterns
- **Built-in Sound Kit** — Kick, Snare, Hi-Hat, Open Hat, Clap, Tom, Perc, Bass (all synthesized, no samples needed)
- **Transport Controls** — Play, pause, stop, BPM (30-300), swing
- **Mixer** — Per-track volume faders, mute, solo
- **Real-time** — Sequencer responds to changes instantly during playback

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and hit play.

## Tech Stack

Next.js 16 + TypeScript, Tailwind CSS v4, Tone.js, Zustand v5.
